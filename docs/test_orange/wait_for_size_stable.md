
# `wait_for_size_stable(locator: Locator, stable_duration=0.5, poll_interval=0.25, timeout=3.0)`

## Назначение

Функция `wait_for_size_stable` используется для ожидания **стабильного размера элемента** на странице.  
Она не требует знания ожидаемой ширины или высоты и работает для любых элементов с динамическим или анимированным layout.

Используется для:

- проверки окончания анимаций и переходов размеров элементов  
- диагностики динамического layout  
- предотвращения flaky-тестов при изменяющихся размерах UI  

---

## Сигнатура функции

```python
def wait_for_size_stable(
    locator: Locator,
    stable_duration: float = 0.2,
    poll_interval: float = 0.05,
    timeout: float = 3.0
) -> dict[str, float]:
````


## Параметры

* `locator` (`Locator`) — Playwright Locator, указывающий на элемент, размеры которого отслеживаются.
* `stable_duration` (`float`, по умолчанию `0.2`) — время в секундах, которое размеры должны оставаться неизменными, чтобы считаться стабильными.
* `poll_interval` (`float`, по умолчанию `0.05`) — частота проверки размеров элемента (в секундах).
* `timeout` (`float`, по умолчанию `3.0`) — максимальное время ожидания стабилизации (в секундах).


## Возвращаемое значение

Словарь с текущими размерами элемента:

```python
{
    "width": float,
    "height": float
}
```

Пример:

```python
{
    "width": 83.1875,
    "height": 720
}
```

---

## Как работает

1. Функция получает `element_handle()` для переданного `locator`.
2. В цикле проверяет размеры через `getBoundingClientRect()`.
3. Если размер совпадает с предыдущим, запускается таймер стабильности.
4. Если размер остаётся неизменным в течение `stable_duration`, функция возвращает текущие размеры.
5. Если размер меняется, таймер сбрасывается.
6. Если превышен `timeout`, выбрасывается `TimeoutError`.

---

## Исключения

* `RuntimeError` — если невозможно получить `element_handle()` (элемент отсутствует в DOM или скрыт).
* `TimeoutError` — если размеры не стабилизировались в течение `timeout`.

---

## Пример использования

```python
sidepanel = page.get_by_role("navigation", name="Sidepanel")

# Дожидаемся стабилизации размера бокового меню
size = wait_for_size_stable(sidepanel)

print(f"Width: {size['width']}, Height: {size['height']}")
```

---

## Преимущества

* Не зависит от конкретного ожидаемого значения размеров.
* Подходит для элементов с анимациями и динамическими layout.
* Уменьшает вероятность flaky-тестов, связанных с временными изменениями размеров.
* Настраиваемая стабильность (`stable_duration`) и частота проверки (`poll_interval`).

---

## Рекомендации

* Использовать для элементов, которые изменяют размер динамически (например, боковые панели, модальные окна, раскрывающиеся карточки).
* Для статичных элементов или когда известен точный размер можно использовать более простые проверки через `expect(locator).to_have_css()` или `element_size()`.
